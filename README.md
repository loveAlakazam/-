# 유형별 남몰래 알고리즘

## 자료구조 활용

- 배열(1차원/ 2차원 배열)
- 스택
- 큐
- linked list
- hash
- tree

  - Trie (현재 라인 문제에 자주나오는 빈출)
- heap
